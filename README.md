# HotSpotManager-HSM

What this is:

It's an iOS Shortcut that makes it easier to set up hotspots, you can:

- Create WORKING QR-Codes that let others join your hotspot
  
- set/change your hotspot password
  
- Turn your hotspot on/off (wow)
